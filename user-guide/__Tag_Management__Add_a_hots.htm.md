###  Add a hotspot

You can create additional hotspots for existing tags.Â

  1. In the Tags List, locate the tag that you want to add a hotspot to. 

A number in brackets beside the tag indicates the number of hotspots it
already has. If there is no number, that tag currently has no hotspots.

  1. Hover over the tag name and click on the blue Add Hotspot button. 
  2. Move the hotspot to the position on the document where you want to add it.Â 
  3. To change the size of the hotspot, hold down the ALT key while dragging. 
  4. When the hotspot is in the correct position, click to place it and press OK.
